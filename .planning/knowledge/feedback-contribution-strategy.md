# Contribution Strategy

> Updated: 2026-03-31

## Current Approach
Work locally on a fork. Accumulate improvements across sessions. Don't create PRs immediately — let changes meld and mature. When ready, propose changes upstream in well-scoped batches with context on WHY.

- Group by feature area, not by session
